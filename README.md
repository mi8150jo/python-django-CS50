# 飛行機のフライトを管理・予約するWebアプリ
CS50で学んだ内容です。<br>
## 概要
・作成したアカウントのみログインしてフライトを管理する<br>
・フライトに搭乗者を登録できる<br>
・ユーザーが認証されていない場合ログイン画面にリダイレクトする<br>
## 機能
### 管理者
・アカウントの登録と削除<br>
・フライト情報の登録と削除<br>
・搭乗者の登録と削除<br>
### ユーザー
・フライトに搭乗者の追加<br>
・フライト情報の詳細情報の確認<br>
## 開発環境<br>
・Python 3.11.0<br>
・Django 3.0.7<br>
・MySQL3<br>
